# <%= projectName %>

### Set up the project

#### Create a Python virtual environment

```bash
python3 -m venv env
source env/bin/activate
```

#### 📦 Install the requirements

```bash
curl -L https://raw.githubusercontent.com/software-mansion/protostar/master/install.sh | bash
```
Then restart the terminal and run the command:
```bash
protostar install
```

### ⛏️ Compile

```bash
protostar build
```

### 🌡️ Test

```bash
# Run all tests
protostar test
```
